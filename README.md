# AIM Report — Audience Intelligence Metrics Report

> 인터랙티브 AIM 리포트 대시보드

---

## 개요

AIM(Audience Intelligence Metrics) 리포트는 키워드 데이터를 마케팅 퍼널 단계별로 분석하여, 광고주·대행사·렙사에 실질적인 인사이트와 실행 전략을 제공하는 웹 기반 리포트 시스템입니다.

입력 데이터 조합에 따라 리포트 구성이 동적으로 달라집니다.

---

## 리포트 구성

### 공통 (기본 AIM 리포트 — 키워드 데이터만 있을 때)

| 섹션 | 설명 |
|---|---|
| **AIM 점수 요약** | 전체 기간 퍼널별 AIM 점수 한눈에 보기 |
| **월별 AIM 점수 바차트** | 월별 점수 추이. Peak/최저 월 클릭 시 AI가 외부 이슈 해석 제공 |
| **마케팅 퍼널별 월별 점수** | 5단계 퍼널 × 월별 상세 점수 |
| **퍼널별 키워드 클라우드** | 각 퍼널을 대표하는 키워드 시각화 |
| **AI 인사이트** | 전체 데이터를 종합한 AI 자동 해석 |
| **다음 실행 전략** | AI가 제안하는 퍼널별 마케팅 액션 플랜 |

---

## 마케팅 퍼널 5단계

```
1. 인지 → 2. 탐색 → 3. 의도 → 4. 유입 → 5. 확산
```

| 단계 | 의미 | 대표 키워드 성격 |
|---|---|---|
| **1. 인지** | 브랜드/카테고리 최초 인식 | 브랜드명, 카테고리 일반어 |
| **2. 탐색** | 정보 탐색 및 비교 | 리뷰, 비교, 추천 |
| **3. 의도** | 구매/전환 의도 형성 | 가격, 구매, 할인 |
| **4. 유입** | 실제 사이트/채널 유입 | 공식사이트, 바로가기 |
| **5. 확산** | 경험 공유 및 바이럴 | 후기, 인증, 공유 |

## 조건별 리포트 확장 구성

### + 소재 데이터 포함 시
- 집행 소재와 키워드 데이터 교차 분석
- 소재별 퍼널 기여도 측정
- 소재 관련 키워드 노출 여부 분석
- 소재-키워드 매칭 갭 리포트 (어떤 소재가 어떤 퍼널을 자극했는지)

### + 캠페인 성과 데이터 포함 시
- 캠페인 KPI(노출, 클릭, CTR, CVR, ROAS 등)와 AIM 점수 상관관계 분석
- 퍼널 단계별 성과 기여도 분석
- 성과 지표 유형에 따라 분석 항목 자동 조정
  - 인지 캠페인: 노출/도달 중심
  - 퍼포먼스 캠페인: CTR/CVR/ROAS 중심

### + 경쟁사 데이터 포함 시
- 경쟁사 대비 퍼널별 AIM 점수 비교
- 경쟁사 키워드 점유율 분석 (Share of Search)
- 퍼널별 경쟁 강도 히트맵
- 경쟁사 대비 기회 키워드 발굴
- 자사 우위/열위 영역 AI 해석

## 기술 스택

| 영역 | 기술 |
|---|---|
| **UI 프레임워크** | HTML5 / CSS3 / Vanilla JS |
| **폰트** | Paperlogy (페이퍼로지) |
| **차트** | Chart.js |
| **키워드 클라우드** | D3.js / WordCloud2.js |
| **AI 해석** | OpenAI API (GPT-4o) |
| **배포** | 정적 웹 호스팅 (S3 + CloudFront 권장) |

## 데이터 입력 형식

### 키워드 데이터 (필수)
```json
{
  "brand": "브랜드명",
  "period": "2024-01 ~ 2024-12",
  "keywords": [
    {
      "keyword": "키워드",
      "funnel": "인지|탐색|의도|유입|확산",
      "month": "2024-01",
      "search_volume": 12000,
      "aim_score": 78.5
    }
  ]
}
```

### 캠페인 성과 데이터 (선택)
```json
{
  "campaigns": [
    {
      "month": "2024-01",
      "impressions": 1200000,
      "clicks": 45000,
      "ctr": 3.75,
      "conversions": 1200,
      "cvr": 2.67,
      "roas": 380
    }
  ]
}
```

### 소재 데이터 (선택)
```json
{
  "creatives": [
    {
      "id": "CRV_001",
      "name": "소재명",
      "type": "image|video|banner",
      "month": "2024-01",
      "keywords_included": ["키워드1", "키워드2"],
      "funnel_target": "인지"
    }
  ]
}
```

### 경쟁사 데이터 (선택)
```json
{
  "competitors": [
    {
      "brand": "경쟁사명",
      "keywords": [
        {
          "keyword": "키워드",
          "funnel": "인지",
          "month": "2024-01",
          "search_volume": 8000,
          "aim_score": 65.2
        }
      ]
    }
  ]
}
```

## 주요 기능 상세

### 월별 AIM 점수 바차트 인터랙션
- Peak 월 또는 최저 월 클릭 시 → AI가 해당 월의 외부 이슈 분석
- 이슈가 없는 월은 팝업 미노출
- 예: *"2024년 3월은 경쟁사 신제품 출시 및 업계 이슈로 인지 점수가 급등했습니다"*

### AI 인사이트 섹션
- 전체 퍼널 흐름의 강점/약점 해석
- 이상 패턴(급등/급락) 자동 감지 및 원인 추론
- 브랜드 성장 단계 진단

### 다음 실행 전략
- 퍼널별 취약 구간에 대한 구체적 액션 제안
- 소재/캠페인 데이터 있을 경우 연계 전략 포함

---

## 사용 방법

1. `data/` 폴더에 분석 데이터 JSON 파일 세팅
2. `index.html`을 브라우저에서 열거나 웹서버로 실행
3. 상단 옵션에서 포함할 분석 항목 선택 (소재 / 캠페인 / 경쟁사)
4. 리포트 자동 생성
